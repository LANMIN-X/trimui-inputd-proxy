# Trimui_inputd proxy

一个用于 **TrimUI Smart Pro** 的输入代理程序。  
通过 **uinput** 创建虚拟手柄，对物理手柄输入进行转发，并支持 **Force Feedback（震动）**。

---

## 功能

- 转发物理手柄输入事件
- 支持 **FF 震动（Force Feedback）**
- 通过 GPIO 控制真实震动马达

---

## 使用说明

- 程序运行后会独占物理手柄输入设备
- 所有输入事件将被转发到虚拟手柄
- 游戏或应用只需识别虚拟手柄即可

---

## 已知限制

- 部分应用无法使用
- 部分游戏可能无法正常工作
- 尤其是以下情况：
  - 游戏已自行创建虚拟手柄
  - 游戏或前端已将手柄映射为键盘
  - 存在多层输入代理或重复虚拟设备

上述情况下，可能出现输入或震动异常。
